Checklist de Verificación Bloom Box SPA

Tiempo estimado: 02:00

Pasos de Verificación

Paso 1: Abrir consola del navegador

Presiona F12 → Ve a las pestañas Network/Red y Console

Pendiente

Paso 2: Verificar CSRF

Carga la home y verifica que registro.php?action=csrf responde 200 con {"csrf": "..."}

Pendiente

Paso 3: Verificar registro

Haz el registro: el POST a registro.php debe devolver {"ok": true, "participant_id": N}

Pendiente

Paso 4: Solucionar errores 419/403

Si devuelve 419/403 → es CSRF o cookie de sesión. Revisa que estás bajo https://TU-DOMINIO/bloom_box_spa/...

Pendiente

Paso 5: Verificar juego

Al hacer clic en una caja, revisa el POST a juego.php → debe responder {"ok": true, "prize": { ... }}

Pendiente

Paso 6: Verificar base de datos

Verifica en MySQL:
SELECT COUNT(*) FROM participantes;
SELECT COUNT(*) FROM ganadores;
SELECT * FROM ganadores ORDER BY id DESC LIMIT 5;

Pendiente
