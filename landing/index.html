Sobre el Proyecto

Bloom Box Spa fue desarrollado como una plataforma completa con frontend, backend, base de datos y panel administrativo.

Backend PHP

La versión completa ejecuta registro, sesiones, validaciones, premios y ganadores mediante PHP.

Base de Datos MySQL

Participantes, premios, ganadores y usuarios administrativos se gestionan desde MySQL.

Seguridad

Incluye protección CSRF, sesiones, control de acceso y consultas preparadas.
