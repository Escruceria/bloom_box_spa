# 🌸 Bloom Box Spa

Sistema web de sorteos promocionales con registro de participantes, juego interactivo de cajas sorpresa, gestión de premios, administración de ganadores y panel administrativo completo.

Bloom Box Spa es una plataforma web desarrollada en PHP, MySQL, JavaScript, HTML5 y CSS3 para la gestión de sorteos promocionales.

El sistema permite el registro de participantes, validación de usuarios, juego interactivo de selección de premios mediante cajas sorpresa, administración de ganadores y gestión completa de premios desde un panel administrativo seguro.

Características principales:

• Registro de participantes con validaciones frontend y backend.
• Protección CSRF y controles básicos de seguridad.
• Juego interactivo de cajas sorpresa con animaciones y efectos visuales.
• Asignación y control de premios desde base de datos.
• Gestión de participantes, premios y ganadores.
• Exportación de reportes en Excel y CSV.
• Panel administrativo con autenticación y control de acceso.
• Diseño responsive compatible con computadores, tablets y dispositivos móviles.
• Implementación sobre Apache (XAMPP), PHP 8.x y MySQL.
• Soporte para despliegue mediante dominio DuckDNS y certificados SSL.

Tecnologías utilizadas:
- PHP 8.x
- MySQL
- JavaScript (Vanilla JS)
- HTML5
- CSS3
- Apache
- XAMPP

Proyecto desarrollado como solución integral para campañas promocionales y sorteos digitales de Bloom Box Spa.
